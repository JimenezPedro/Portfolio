Comandos Linux

Guía rápida de los comandos más usados en Linux

Categorías

Navegación
  • pwd → Muestra la ruta actual
  • ls → Lista archivos en el directorio
  • cd [ruta] → Cambia de directorio
Archivos y Directorios
  • touch archivo.txt → Crear archivo vacío
  • mkdir carpeta → Crear carpeta
  • rm archivo → Eliminar archivo
  • rm -r carpeta → Eliminar carpeta
  • cp origen destino → Copiar archivos
  • mv origen destino → Mover/renombrar
Procesos
  • ps aux → Lista procesos activos
  • top → Monitor en tiempo real
  • kill -9 PID → Finaliza un proceso
Permisos
  • chmod 755 archivo → Cambia permisos
  • chown usuario:grupo archivo → Cambia propietario
Red
  • ping google.com → Verificar conectividad
  • ifconfig / ip a → Mostrar interfaces de red
  • curl url → Obtener datos de una URL
  • scp archivo usuario@host:/ruta → Copiar por SSH
Utilidades
  • man comando → Manual del comando
  • history → Historial de comandos
  • clear → Limpiar la terminal
  • echo "texto" → Imprimir texto

Práctica recomendada: OverTheWire

Una de las mejores formas de aprender Linux y seguridad básica es el juego de retos OverTheWire: Bandit. Aquí tienes un resumen de los primeros niveles y los comandos clave para resolverlos:

Nivel 0 → 1

Conectarse al servidor vía SSH.

  • ssh bandit0@bandit.labs.overthewire.org -p 2220 → Acceder al nivel inicial.
Nivel 1 → 2

Leer un archivo llamado -.

  • cat ./- → Leer un archivo con nombre especial.
Nivel 2 → 3

El archivo está dentro de una carpeta llamada spaces in this filename.

  • cat "spaces in this filename/archivo" → Manejar espacios en nombres.
Nivel 3 → 4

El archivo está oculto.

  • ls -a → Mostrar archivos ocultos.
  • cat .hidden → Leer archivo oculto.
Nivel 4 → 5

El archivo con la clave está en un directorio con muchos archivos.

  • file * → Identificar archivos de texto.
  • cat archivo.txt → Leer la clave.
Nivel 5 → 6

Buscar un archivo con permisos específicos.

  • find . -type f -size 1033c ! -executable → Filtrar por tamaño y permisos.
